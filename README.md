# Personal Finance Dashboard

A responsive Personal Finance Dashboard built using **HTML, CSS, JavaScript, and Chart.js**. The application helps users manage their personal finances by tracking income, expenses, savings, and financial goals through an interactive and user-friendly interface.

---

## Features

### Dashboard
- View Total Balance, Income, Expenses, and Savings
- Expense Breakdown Chart
- Current Month Cash Flow Chart
- Recent Transactions
- Savings Goal Progress

### Transactions
- Add Transactions
- Edit Transactions
- Delete Transactions
- Duplicate Transactions
- View Transaction Details

### Insights
- Highest Income and Expense
- Average Monthly Spending
- Saving Rate
- Expense Breakdown
- Income vs Expense Analysis
- Monthly Comparison
- Spending Trend
- Category Analysis

### Profile
- Manage Personal Information
- Update Financial Goals
- Manage Preferences

---
